== Description ==
This Plugin provides a widget that allows the user to display actual bargains (Schnaeppchen) in the sidebar. You can choose the number of articles to show and where you get them from (reichweite.de or schnaeppchenforum.com). 


== Installation ==
The installation of this plugin is simple. 

1. Upload the 'schnaeppchen-widget' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to the 'Widgets' submenu in the 'Design' menu
1. Move the Widget to the sidebar via drag&drop

== Frequently Asked Questions ==
Q: Why should I use this plugin?<br />
A: It allows you to have new, fresh and interesting content on your blog, which brings you more views. Your Users will be happy making bargains. 

Q: Is the use of this plugin/widget free?<br />
A: Yes, it's free. You don't have to pay neither for the widget nor for the content shown.

Q: How often do you update the offers?<br />
A: The offers are updated several times a day, sometimes even ten times a day.
